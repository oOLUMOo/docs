# Relational Database Service (RDS)

**vDB Relational Database Service (RDS)** có nhiều loại cơ sở dữ liệu quan hệ quen thuộc với bạn như:

* MySQL: 5.5, 5.7, 8.0.
* Mariadb: 10.1, 10.3, 10.4, 10.5.
* Postgresql: 10, 12, 13, 15.
