# MemoryStore Database Service (MDS)

**vDB MemoryStore Database Service (MDS)** gồm các hệ lưu trữ dữ liệu trên bộ nhớ (in-memory) quen thuộc với bạn như:

* Redis: 3.2, 4.0, 5.0, 6.2.
* Memcached (coming soon)
